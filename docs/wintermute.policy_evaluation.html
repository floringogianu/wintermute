wintermute.policy_evaluation.exploration_schedules module¶

Various exploration schedules.

  • constant_schedule(value)

    constant_schedule(.1) => .1, .1, .1, .1, .1, …

  • linear_schedule(start, end, steps_no, warmup)

    linear_schedule(.5, .1, 5) => .5, .4, .3, .2, .1, .1, .1, .1, … linear_schedule(.5, .1, 5, 3) => .5, .5, .5, .5, .4, .3, .2, .1, …

  • log_schedule(start, end, steps_no)

    log_schedule(1, 0.001, 3) => 1., .1, .01, .001, .001, .001, …

Returns a constant, linear or logarithmic scheduler.

name (str, optional): Defaults to “linear”. Schedule type. start (int, optional): Defaults to 1. Start value. end (float, optional): Defaults to 0.01. End value. steps (int, optional): Defaults to 0. No of steps during which the

value is degraded towards its end value.

warmup_steps (int, optional): Defaults to 0. No of steps during which

the schedule remains constant at the start value.

Returns

A schedule for a given value.
